## 🧩 프로젝트 소개

물듬은 전공동아리를 더 쉽고 효율적으로 운영할 수 있게 도와주는 플랫폼입니다.
선생님과의 소통은 더 편리하게, 공지와 정보 전달은 더 감각적으로.
전공동아리 운영에서 느꼈던 모든 불편함을 물듬이 해결합니다.
